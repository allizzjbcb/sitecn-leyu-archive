# sitecn-leyu-archive

本仓库用于归档和发布多个独立的 HTML 页面。页面内容来自网络公开信息，以静态文件形式存放，便于查阅和长期保存。

## 项目简介

sitecn-leyu-archive 是一个静态 HTML 页面归档仓库。仓库中收录的页面均为独立的 HTML 文件，不依赖外部数据库或后端服务。页面内容以原始形式保存，不做修改或重新编排。

## 目录说明

仓库根目录下直接存放 HTML 文件。每个文件独立对应一个归档页面。目录结构如下：

```
/
├── index.html
├── page-example-001.html
├── page-example-002.html
└── ...
```

页面文件名采用英文或数字命名，便于索引。部分页面可能附带同名的资源文件夹（如 `assets/` 或 `images/`），用于存放页面所需的图片、样式表或脚本文件。

## 页面归档说明

- 所有页面均为离线可用的静态 HTML 文件。
- 页面内容保持原始状态，不做任何删改或注释。
- 归档时间以文件提交日期为准。
- 部分页面可能包含外部资源链接，这些链接可能随时间失效。
- 页面内容仅供存档参考，不代表本仓库维护者的立场或观点。

## 维护说明

本仓库不定期更新。新增页面会以独立提交的形式加入，并在提交信息中注明页面主题或来源说明。不提供定期更新承诺。

### 如何贡献

欢迎通过 Issue 或 Pull Request 提供建议或提交新的归档页面。提交时请确保：

- 页面内容为独立的 HTML 文件，无外部依赖（或已附带必要资源）。
- 提交信息简洁明了，说明页面主题或归档原因。
- 不包含违反 GitHub 服务条款的内容。

### 注意事项

- 本仓库不提供页面内容的真实性或时效性保证。
- 页面内容可能涉及第三方信息，请自行判断其准确性。
- 如有版权问题，请通过 GitHub 联系仓库维护者，我们会及时处理。

## 许可

本仓库的归档页面内容遵循其原始版权声明。无特别声明的页面，按 GitHub 默认条款处理。仓库本身的组织及描述文本采用 MIT 许可。
